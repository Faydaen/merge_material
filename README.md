При нажатии на кнопку Material info в material_info (вкладка Scripting) создается json в котором данна информацию о том какие материалы как будут мержится.
Список бледоровских объектов (навзвание мешей), в каждом из котроых лежить список объектов в формате 

```
{
  "list": список слотов которые должны будут быть смережны
  "target": строка с новым назавием слота 
}
```
Этот файл можно подредактировать, затем нажать на Merge material чтобы смержить одинаковые материалы
