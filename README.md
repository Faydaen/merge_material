# Merge Material
Проект для объединения одинаковых матеиалов. Полезен при импорте из daz studio

## Как пользоваться
Нужно выдлеить объект на кнопку Material info
В material_info (вкладка Scripting) создается json в котором данна информацию о том какие материалы как будут мержится.
Этот json имеет следующий формат:

Список объектов blender (навзвание мешей), в каждом из котроых лежить список объектов в формате 
```
{
  "Название первого меша": список информации для мерджа материалов первого меша,
  "Название второго меша": список информации для мерджа материалов второго меша
  ...
}
```

В каждом списке информации для мерджа материалов лежат объекты в вот таком формате
```
{
  "list": список слотов которые должны будут быть смережны,
  "target": строка с новым назавием слота 
}
```
Этот файл можно подредактировать, затем нажать на Merge material чтобы смержить одинаковые материалы
